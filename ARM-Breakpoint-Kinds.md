#### E.5.1.1 ARM Breakpoint Kinds


These breakpoint kinds are defined for the '`Z0`' packets.

> 这些断点类型是为'Z0'数据包定义的。

[2]


:   16-bit Thumb mode breakpoint.

> 16位Thumb模式断点。

[3]


:   32-bit Thumb mode (Thumb-2) breakpoint.

> 32位Thumb模式（Thumb-2）断点。

[4]


:   32-bit ARM mode breakpoint.

> 32 位 ARM 模式断点。
